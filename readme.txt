cdr格式:
   呼叫标志,呼叫组,呼叫线路,主叫号码,被叫号码,主叫外显,被叫外显,创建时间,接通时间,挂断时间,sessionID,企业号,BillingCode,呼叫类型，保留字段
1. 故障话单: 创建时间 == 接通时间 == 挂断时间
2. 故障=1 非故障=0
3. 记录维度: //初版不实现
   -时间线
   -主叫号码
   -被叫号码
   -主叫外显
   -被叫外显
   -企业号
4. 存储：bitmap/kv, 仅保留当天数据
5. 统计：平台 / 话务商 / 地域(外显号) 
6. 告警：阈值设定
7. 依赖：
   外显号 ==> 话务商
   外显号 ==> 地域  
8. 结构：
   [umg_1/cdr] -- |
   [.........] -- | =采集=> [cdrAlarm] =告警=> [公司]
   [umg_n/cdr] -- |
